# Claude Cowork ワークショップ

AI活用の第一歩を体験する30分間のハンズオンワークショップ教材です。

## ワークショップ概要

- **対象**: AI活用に興味がある方（プログラミング不要）
- **所要時間**: 30分
- **内容**: Claude Coworkを使ったビジネスタスクの自動化体験

### プログラム

1. Claudeの基礎知識（モデル・使い方）
2. 課題1: Web検索でハンバーガー3社比較 → xlsx + docx作成
3. 課題2: アンケート50件の横断処理 → docx作成
4. 自分の業務への応用を考える

## リポジトリ構成

```
docs/index.html           ← 投影用ガイド（GitHub Pages公開）
download/cowork_workshop.zip ← 参加者配布用（課題フォルダ+prompt.md）
worksheet.docx            ← 運営側ワークシート（A4 1枚）
```

### 参加者配布zip構成

```
cowork_workshop/
├── 課題①/
│   └── prompt.md          ← 課題①のプロンプト
└── 課題②/
    ├── prompt.md          ← 課題②のプロンプト
    └── survey_responses/  ← アンケート50件
```
